
طريقة النشر على Render (مجاني):
1) افتح GitHub وأنشئ مستودع جديد (Private أو Public).
2) ارفع ملفات هذا المجلد كما هي (يشمل app.py و Procfile و requirements.txt و render.yaml).
3) ادخل Render → New + → Web Service → Connect GitHub → اختر المستودع.
4) سيقرأ Render ملف render.yaml تلقائياً. اضغط Create Web Service.
5) عند الانتهاء سيظهر لك رابط مباشر مثل: https://bookstore-accounting.onrender.com
6) أول فتح سيحولك إلى صفحة الإعدادات لإدخال (اسم المكتبة/الهاتف/العنوان) ورفع الشعار.
